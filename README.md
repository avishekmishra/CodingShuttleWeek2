# CodingShuttleWeek2
This contain example to Create rest controller with 

1. Clean Architecture and MVC, 
2. Exception handling,Different Response Code, Custom response
3. H2 in memory database
4. We used ModelMapper to map from DTO object to Entity and Entity object to DTO.
5. We used Jakarta validation (<artifactId>spring-boot-starter-validation</artifactId>) for validating request.
